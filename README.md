LiterAlura
--------
LiterAlura é uma aplicação desenvolvida com Java e Spring Boot que funciona como um catálogo de livros integrado à API Gutendex. O sistema permite buscar livros pelo título, armazenar os dados em um banco PostgreSQL e consultar informações registradas no banco.

-------------------------

*Tecnologias Utilizadas*

Java 17+

Spring Boot

Spring Data JPA

PostgreSQL

Maven

API Gutendex

-------------

*Funcionalidades*

Buscar livros pelo título na API Gutendex

Inserir livros no banco de dados

Listar livros registrados

Listar autores cadastrados

----------------

*Estrutura do Projeto*

dto: classes responsáveis por mapear os dados da API

model: entidades JPA

repository: interfaces de acesso ao banco de dados

service: regras de negócio e consumo da API

resources: configurações da aplicação

Listar livros por idioma

Listar autores vivos em determinado ano
